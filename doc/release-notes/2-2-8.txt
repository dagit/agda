------------------------------------------------------------------------
-- Release notes for Agda 2 version 2.2.8
------------------------------------------------------------------------

Important changes since 2.2.6:

Language
--------

* The injectivity of type constructors is disable by default.

  To enable the injectivity of type constructors give the flag
  --injective-type-constructors on the command line or (recommended)
  as an OPTIONS pragma.

  N.B. This flag makes Agda anti-classical and possibly inconsistent.

  See test/succeed/InjectiveTypeConstructors.agda for an example.
